# Product Overview

## Database Control Panel

A secure, private web application for managing multiple PostgreSQL databases from a single dashboard interface. Designed specifically for teams managing multiple websites deployed on Coolify.

## Core Purpose

Eliminates the need to SSH into servers or use command-line tools for database management by providing:

- **Centralized Management**: View and manage all PostgreSQL databases from one interface
- **Safe Editing**: Read-only by default with explicit edit mode activation and mandatory confirmations
- **Audit Trail**: Complete logging of all database changes with before/after data
- **Production Safety**: Extra confirmation requirements and visual warnings for production databases
- **Role-Based Access**: Admin and Viewer roles with appropriate permissions

## Key Features

- Multi-database registry with encrypted credential storage
- Visual table browser with pagination and search
- Safe edit system with change previews and confirmations
- Complete audit logging for compliance and debugging
- Project organization for grouping related databases
- Environment classification (dev/staging/prod) with appropriate safety measures

## Target Users

DevOps teams, database administrators, and developers who need secure, convenient access to multiple PostgreSQL databases without command-line complexity.
